大井町発車

東急大井町線をご利用くださいまして、ありがとうございます。この電車は二子玉川行きです。次は下神明、下神明です。お出口は左側です。途中の戸越公園では、後ろの2両扉が開きません。戸越公園でお降りのお客様は、前の車両にお移りください。

Thank you for using the Tōkyū Ōimachi Line. This train is bound for Futako-tamagawa. The next station is Shimo-shimmei.

The next station is Futako-tamagawa. This is the last stop of this line.

二子玉川到着

まもなく二子玉川、二子玉川、終点です。田園都市線ご利用のお客様は乗り換えです。中央林間方面にお出でのお客様は1番線、渋谷・半蔵門線方面にお出でのお客様は4番線でお待ちください。お出口は左側です。東急大井町線をご利用くださいまして、ありがとうございました。

We will soon make a brief stop at Futako-tamagawa. This is the last stop of this line. Passengers changing to the Den-en-toshi Line, please transfer at this station. Thank you for using the Tōkyū Ōimachi Line.
